# Site da Serralheria Tecno Ferro

Site desenvolvido em HTML e CSS para apresentar os serviços, portfólio e formas de contato da serralheria.  
Totalmente responsivo e otimizado para dispositivos móveis.

## Tecnologias usadas
- HTML
- CSS

## Funcionalidades
- Seção de serviços
- Galeria de portfólio
- Hero com CTA
- Layout moderno e responsivo.
