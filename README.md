# Site-Serralheria-Tecno-Ferro
Website da serralheria desenvolvido em HTML e CSS, com foco em apresentar serviços, portfólio de projetos, diferenciais e formas de contato. Projeto leve, moderno e responsivo, pensado para facilitar o acesso dos clientes e destacar a qualidade dos trabalhos realizados.

## Tecnologias usadas
- HTML
- CSS

## Funcionalidades
- Seção de serviços
- Galeria de portfólio
- Hero com CTA
- Layout moderno e responsivo.
